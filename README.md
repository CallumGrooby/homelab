# Homelab

A self-hosted homelab built on a HP EliteDesk running multiple services. Designed as a hands-on learning environment for Linux administration, networking, containerisation, and self-hosted application deployment.

## Skills Demonstrated

- Linux server installation and administration (Ubuntu Server)
- Remote server management via SSH
- Docker container deployment and management
- Self-hosted application configuration
- Network security — tunnelling over port forwarding (playit.gg)
- Physical rack build and hardware configuration
- File sharing and network storage (SMB/CasaOS)
